OOblee

This apps helps you find live music events inside the United States. Once an event is selected it provides the location of the venue and weather information of the city for the next ten days. I built this project with my partners Anita Ford and Zachary Bird.


I was mostly responsible for the front end. The page uses bootstrap to be displayed and is phone responsive. I wanted to get out of the squareness so I learn the way to create the radius on the header divs. I believe the most remarkable part of the project was the level of collaboration we had as a team. Despite we did not know each other we were able to communicate and solve together the challenges that we each one face.
