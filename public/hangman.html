Hangman

This is a simple hangman game with a list of cities in Germany. As an elementary teacher, tt was hard to create a game with the picture of a man being hanged. For that reason, I put an abstract representation of a man. The man is a column of white rectangles than turn bloody red each time the player misses a letter.


This was the first homework I built. At that time, I was upset after learn Bootstrap. I thought Bootstrap might limit my ability to create so I look for a design that were very hard to do with Bootstrap. It was a little challenging to display the word because my jQuery skills were very limited.


Each letter is displayed inside a pink div that matches the color of the background. Each pink div is displayed inside a white div. The width of the white div is adjusted to accommodate the number of letter in the word. I use CSS flex-box to equally distribute the pink boxes and make the white div looks as a bookcase.
